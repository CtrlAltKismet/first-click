Learn Essential Digital Skills

Explore beginner-friendly lessons designed to help you build confidence with technology. Each topic below gives you a quick introduction before the full lesson launches.

Learning Topics

Mouse & Keyboard Skills

The mouse and keyboard are the main tools used to control a computer. Getting comfortable with clicking, scrolling, and typing builds confidence quickly.

Definition: A mouse is a pointing device used to select things on-screen. A keyboard is used to type letters, numbers, and shortcuts.

Tip: Practise right-clicking an icon and reading the menu options. It teaches you how computers offer choices.

Internet & Internet Safety

The internet lets you browse websites, watch videos, and use online services. Learning basic safety helps you avoid scams and risky sites.

Definition: The internet is a global network that connects devices so information can be shared online.

Tip: Look for https:// in the web address and a padlock icon before entering personal details.

Hardware vs Software

Computers need physical parts and programs to work. Knowing which is which helps you describe problems more clearly (and fix them faster).

Definition: Hardware is the physical equipment (screen, keyboard, laptop). Software is what runs on it (apps, websites, operating systems).

Tip: If you can touch it, it’s hardware. If it’s on the screen, it’s software.

Email & Email Safety

Email is used for messages, appointments, and attachments. Learning the basics helps you communicate clearly and avoid phishing attempts.

Definition: Email (electronic mail) is a way to send digital messages over the internet.

Tip: If an email pressures you to “act now” or “confirm details”, slow down and double-check the sender.

Viruses & Computer Safety

Computers can be infected by harmful software (malware). Basic habits like updates and careful downloads reduce risk massively.

Definition: A virus is a type of malicious software designed to spread and cause damage or steal information.

Tip: Keep your computer updated and avoid downloading files from unfamiliar websites.
